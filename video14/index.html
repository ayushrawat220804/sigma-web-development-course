CSS Introduction

CSS stands for Cascading Style Sheets.

HTML helps to create the skeleton (bone structure) of a webpage, but CSS makes it beautiful.

CSS helps to style HTML files to make them look attractive. Some tags like the <font> tag are deprecated.
